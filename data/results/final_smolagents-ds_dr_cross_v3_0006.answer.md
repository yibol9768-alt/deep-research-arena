## Priority-Ordered Shopping List for $400 Budget

Assuming the user already has a PC (or budget for PC itself is separate), we focus on peripherals and accessories. Budget: $400.

**Priority Order (most essential first):**

1. **Logitech G502 HERO Gaming Mouse** – $49.99
   - Reason: Essential for precision; highly rated and community-recommended.
2. **Redragon K552 Mechanical Keyboard** – $34.99
   - Reason: Affordable mechanical keyboard; good reviews and community mentions.
3. **Acer SB220Q 21.5\" Monitor** – $129.99
   - Reason: If user doesn't have a monitor, this is a must. Otherwise, skip and allocate funds elsewhere.
4. **Xbox Wireless Controller** – $59.99
   - Reason: Nice-to-have for controller-friendly games; plug-and-play on Windows.
5. **HyperX Cloud II Gaming Headset** – $79.99
   - Reason: Community warns about value, but if user needs a headset for chat, consider alternatives (e.g., Philips SHP9500 + Vmoda Boompro) which may cost similar.

**Budget Calculation:**
- Mouse + Keyboard + Monitor + Controller + Headset = $49.99 + $34.99 + $129.99 + $59.99 + $79.99 = $354.95
- Remaining budget: $45.05 for taxes/shipping or additional accessories.

**Alternative if monitor not needed:**
- Mouse + Keyboard + Controller + Headset = $49.99 + $34.99 + $59.99 + $79.99 = $224.96
- Use remaining $175.04 for a better monitor upgrade or savings.

## Conclusion

Upgrading from console to PC gaming on a budget requires prioritizing input devices (mouse, keyboard) and display. The shopping list above balances essential functionality with community trust, ensuring a smooth transition without breaking the bank.
